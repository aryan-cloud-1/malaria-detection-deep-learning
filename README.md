# Malaria Detection using Deep Learning

This project uses a Convolutional Neural Network (CNN) to detect malaria-infected blood cells from microscopic images.

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Matplotlib

## Features
- CNN model trained on malaria dataset
- Image classification (Parasitized vs Uninfected)
- Testing with new blood cell images

## Dataset
NIH Malaria Cell Images Dataset

## How to Run
1. Install dependencies
2. Run training:
   python train.py
3. Test with an image:
   python test.py
